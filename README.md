# ordered-roulette
 Ordered class roulette: spin to pick names one at a time and copy the order to Google Sheets.

A spinning wheel roulette that picks names one at a time and keeps track of the draw order. Great for classrooms, meetings, or anywhere you need to randomize a sequence.

## Live Demo

https://nlily.github.io/ordered-roulette/

## Features

- Spin the wheel to randomly pick from a list of names
- Names are marked as picked and added to an ordered list (1st, 2nd, 3rd, ...)
- Undo your last pick, reset, or clear the list
- Copy the final order to your clipboard and paste into Google Sheets or Excel

## How to Use

1. Open the live demo link above
2. Enter names in the textarea (one per line) and click Apply
3. Click Spin to pick the next name
4. When finished, use the Copy button to export the order
